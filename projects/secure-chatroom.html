Secure Chatbot

Overview

Secure Chatbot is a simple application that enables the users to communicate through the terminal. The program allows for security manipulation in order to explore vulnerabilities in communication systems.

Features

  • Multiple-user communication
  • Security analysis

Technologies Used

  • Python
  • proxy
  • NumPy
  • MediaPipe (optional)
